# 🛒 Full Stack E-Commerce Website

A full-stack e-commerce web application built using Node.js, Express.js, MongoDB, Mongoose, and EJS.

## 🚀 Features

- User Registration
- User Login
- Product Listing
- Add to Cart
- View Cart
- Remove from Cart
- Checkout

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS

## 📂 How to Run

1. Clone the repository
2. Install dependencies

```bash
npm install
```

3. Start MongoDB

4. Run the project

```bash
node server.js
```

5. Open:

```
http://localhost:3000
```
