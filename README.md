## 📌 About the Project

This project is a beginner-friendly **Full Stack E-Commerce Web Application** developed to understand backend development, database integration, authentication, and dynamic web pages.

It allows users to register, log in, browse products, add items to the cart, remove items, and place an order.

---

## 🚀 Features

- 👤 User Registration
- 🔐 User Login Authentication
- 🛍️ Product Listing
- ➕ Add to Cart
- 🛒 View Cart
- ❌ Remove from Cart
- 💳 Checkout
- 🗄️ MongoDB Database Integration

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/varshinisri14/simple-ecommerce.git
```

Move into the project folder

```bash
cd simple-ecommerce
```

Install dependencies

```bash
npm install
```

Start MongoDB

Run the application

```bash
node server.js
```

Open your browser

```text
http://localhost:3000
```
